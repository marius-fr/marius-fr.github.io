Current setup

So the way I have things set up has solidified some now.

For blogging I currently use Mkdocs, which allows me to keep things pretty tidy, and also allows me to have a structure that I like.

I use GitHub to sync everything between computers, still have some setup left to do with that. Having everything on GitHub will probably make it easier for me in the long run.

Other than that I have Vim set up with a bit too many plugins: - Nerdcomment - Matchit - Nerdtree - vim-airline - Nord-vim theme - vim-polyglot - Neoclide Coc - auto-pairs (buggy for my use) - Rust.vim

Hardware

So my current machine setup has been expanded somewhat. I purchased a old Lenovo Thinkpad X220 to tinker with. I'm not actually sure exactly how old the machine is, but I think its from 2012 or 2013. Other specs are:

  • CPU: i7 something
  • RAM: 8 Gigs
  • HDD: SSD with 120 Gigs of storage
  • 12" screen
  • Extra large battery

So not exactly a beast of a machine compared to today's standard. Battery is at about 50% capacity compared to new, but i still get about 3-4 hours of light use.

Currently I have Debian 12 installed, and running i3wm as a GUI. Again, a pretty simple setup that doesn't require a ton of resources and saves on battery life.

I also have a old mac mini running Debian that acts as a server for anything that might require some more power, since the battery on the X220 is pretty worn out.

Why X220?

The past few months I've been wanting to get back into computers again, particularly some light programming and general tinkering. I already have a Mac Mini (2012-2013?) set up with Debian as a home server, but wanted something portable to use as well.

Having done some research, older Lenovo machines, and the X220 seems well supported by Linux, all the hardware I need to work, worked out-of-the-box with minimal fuzz. The physical size of the machine is small, so its easy to stash anywhere. Awesome keyboard to type on.

Why Debian?

Why not. I'm a old Slackware user, but grew tired of all the work having to be put in to get it to where I wanted it to be. Research before buying anything showed that Debian installed flawlessly on the X220, and everything has worked out of the box. The only issue I have is i3wm not having a easy way to suspend the machine, I'm sure there is a way, I just haven't had time to figure it out yet.
