Vim setup

I've been using Vim for years, quite some time since I used it regularly. Lately I have been trying out different setups and plugins, but it ended up feeling kind of bloated, messy and eventually I lost track of what plugin did what.

So now I've cut down on alot, if not all of the plugins. I currently just have Plugged to manage the two plugins i use, NerdTree and vim-fugitive. Not sure if fugitive is working correctly, but I havent had time to look into it yet.
