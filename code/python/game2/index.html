Finding hits in rows and columns

I have to figure out a way to check each row and column for consecutive blocks in images, i.e.:

      1 1 0 1
    -----
2  1| 1 1 0 1
0  0| 0 0 0 0

And I am having a little trouble figuring it out. The 'correct' solution seems to be to use recursion to parse each row/column and have a look-ahead to see if the next cell has something in it. It might also be possible to just use space for empty image cells, and any value for image, and iterate thru words.

Solution

Recursion would probably be a better(?) solution, but I went with a simpler solution where I just iterate thru the list. Single list is complete, now to figure out how to do it with the double list.
