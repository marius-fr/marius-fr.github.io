(Re)Learning programming

Why Python

I need something easy(?) to start out with, that is well documentet and has a big user mass.

Why rust?

Why not? Its a relatively low-level language, which i always wanted to learn. I also considered Go, but for no obvious reason decided to go with Rust. Python is also still being considered.
